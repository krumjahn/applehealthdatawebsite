OpenClaw + Apple Health Prompts

Copy/paste prompts designed for consistent outputs from the health-analyzer-mac-local skill. These assume your Apple Health export is already imported into the Mac app and the localhost API is running.

Standard prefix (use this in every prompt)

Use the "health-analyzer-mac-local" skill for this request.

This forces OpenClaw to route the request through the local analyzer API.

Daily brief

Use the "health-analyzer-mac-local" skill for this request.
Give me my daily health brief for today and 3 suggestions.

Weekly review

Use the "health-analyzer-mac-local" skill for this request.
Pull my daily brief for the last 7 days and summarize trends in steps, sleep, and workouts.
Then give me a weekly scorecard (Consistency, Recovery, Activity) and 3 experiments for next week.

Sleep consistency

Use the "health-analyzer-mac-local" skill for this request.
For the last 14 days, summarize my sleep consistency. Flag nights with missing sleep data.
Give me 3 high-confidence changes to improve consistency.

Recovery (HRV / resting heart rate)

Use the "health-analyzer-mac-local" skill for this request.
For the last 14 days, summarize my recovery trend using HRV (if available) and resting heart rate.
Call out any anomaly days and plausible causes (sleep, workouts, low activity, missing data).

Training load sanity check

Use the "health-analyzer-mac-local" skill for this request.
Summarize my workouts for the last 7 days and compare to my baseline.
If I'm ramping too fast, suggest a safer plan for the next 7 days.
