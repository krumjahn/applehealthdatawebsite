Changelog

A plain-language timeline of updates to the iOS and Mac app. Most recent first.

2026-04-13
  • Added Dropbox sync — export health data directly to a Dropbox folder using a personal access token.
  • Added Nextcloud / WebDAV sync — upload to any Nextcloud server or WebDAV-compatible storage.
  • Added Home Assistant sync — push health metrics as sensor states using a long-lived access token and configurable entity prefix.
  • Added REST API / Webhook sync — POST a compact JSON health payload to any endpoint; supports POST/PUT/PATCH, Bearer, Basic Auth, and custom headers.
  • Added stable file naming mode — overwrite the same file on each sync instead of creating timestamped copies.
2026-04-11
  • Added Home Screen sync widget for iPhone -- one-tap sync, live progress, and quick export without opening the app.
  • Added widget documentation page with setup guide and troubleshooting.
  • Added Apple Health Widget SEO landing page.
  • Added widget demo video to docs and marketing pages.
2026-03-27
  • Added OpenClaw support for daily Apple Health check-ins.
  • Added a local API for AI tools and developer workflows.
  • Added Mac menu bar sync status.
  • Added direct iPhone-to-Mac WiFi/ethernet sync within local network.
  • Improved iPhone-to-Mac sync and cloud export handling.
  • Improved XML import, saved-analysis loading, and startup responsiveness.
  • Improved subscription reliability and macOS paywall behavior.
2026-02-14
  • Merged and stabilized recent app improvements.
2026-02-13
  • Improved AI model selection so setup is simpler and more flexible.
  • Added support for more smart ring health metrics.
  • Improved pricing messaging: AI analysis is free when using your own API key.
  • Improved release quality checks to reduce regressions.
2026-02-12
  • Improved sleep duration tracking accuracy.
  • Improved import errors to explain what went wrong in clearer language.
  • Fixed large health export files now load more reliably.
2026-02-08

2026-02-05
  • Merged and stabilized recent app improvements.
