AI Analysis

Use AI to find patterns, trends, and anomalies in your Apple Health data. The Mac app supports multiple AI providers -- both cloud-based and fully local -- so you can choose the balance of capability and privacy that works for you.

AI analysis is free when you use your own API key. The app never uploads your health data to our servers. Data goes directly from your Mac to the AI provider you choose.

Supported AI Providers

The Mac app connects to these providers out of the box. You can switch between them at any time.

Provider Type Cost Best For
OpenAI (ChatGPT) Cloud Pay-per-use Strong general analysis, large context windows
Anthropic (Claude) Cloud Pay-per-use Nuanced health reasoning, detailed explanations
OpenRouter Cloud Pay-per-use Access to many models (Llama, Mistral, Gemini, etc.) through a single API key
Ollama Local Free Maximum privacy, no data leaves your Mac
LM Studio Local Free User-friendly local inference with a GUI for model management

Setting Up Each Provider

OpenAI (ChatGPT)

  1. Go to platform.openai.com/api-keys and create an API key.
  2. In the Mac app, open Settings and select OpenAI as your provider.
  3. Paste your API key into the field provided.
  4. Choose a model (e.g., gpt-4o for the best analysis quality, or gpt-4o-mini for faster, cheaper responses).

Anthropic (Claude)

  1. Go to console.anthropic.com/settings/keys and create an API key.
  2. In the Mac app, open Settings and select Anthropic as your provider.
  3. Paste your API key.
  4. Choose a model (e.g., claude-sonnet-4-20250514 for detailed analysis).

OpenRouter

  1. Go to openrouter.ai/keys and create an API key.
  2. In the Mac app, open Settings and select OpenRouter as your provider.
  3. Paste your API key.
  4. Choose from any available model. OpenRouter gives you access to models from many providers (Llama, Mistral, Gemini, and more) through a single key.

Ollama (Local, Free)

  1. Install Ollama from ollama.com.
  2. Pull a model: ollama pull llama3
  3. Make sure Ollama is running on your Mac.
  4. In the Mac app, open Settings and select Ollama as your provider. The app auto-detects the local Ollama server at http://localhost:11434.
  5. No API key is needed. All processing stays on your machine.

LM Studio (Local, Free)

  1. Download LM Studio from lmstudio.ai.
  2. Open LM Studio and download a model (e.g., Llama 3, Mistral, Phi-3).
  3. Start the local server in LM Studio (it runs on http://localhost:1234 by default).
  4. In the Mac app, open Settings and select LM Studio as your provider.
  5. No API key is needed. All processing stays on your machine.

What AI Analysis Can Find

Once you have a provider configured, you can run analysis on your imported Apple Health data. The AI examines your metrics and produces insights that go far beyond what the Health app shows you.

Trends

The AI identifies upward and downward trends in your metrics over time. For example, it might notice that your resting heart rate has been gradually declining over the past three months, or that your step count drops every weekend.

Anomalies

Unusual data points get flagged automatically. A sudden spike in resting heart rate, an unusually short sleep night, or a gap in workout data can all indicate something worth investigating.

Correlations

The AI looks for relationships between different metrics. Does your sleep quality improve on days you walk more than 8,000 steps? Does your HRV drop after late-night workouts? These cross-metric patterns are difficult to spot manually but straightforward for an AI model.

Suggestions

Based on the patterns it finds, the AI generates actionable recommendations. These might include adjusting your bedtime, modifying your workout schedule, or investigating a metric that has changed unexpectedly.

Privacy Considerations

Your choice of provider directly affects where your health data is processed.

Provider Type Data Location Privacy Level
Cloud (OpenAI, Anthropic, OpenRouter) Data is sent to the provider's servers for processing Your data is transmitted over an encrypted connection. Review each provider's data retention policy. The Mac app strips personally identifiable information before sending.
Local (Ollama, LM Studio) Data never leaves your Mac Maximum privacy. No network requests are made. All inference runs on your local hardware.

If privacy is your top priority, use Ollama or LM Studio. Your health data stays entirely on your machine, and no API key or internet connection is required for analysis.
