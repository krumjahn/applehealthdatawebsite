Health Data Export & AI Analyzer

Welcome to the documentation for Health Data Export & AI Analyzer — the privacy-first toolkit for exporting, syncing, importing, converting, and analyzing your Apple Health data across iPhone and Mac.

All data processing happens locally on your devices. Your health data is never uploaded to any server we operate.

What Can You Do?

Getting Started

Set up the iPhone app, choose a sync path, import on Mac, and run your first analysis.

iPhone-to-Mac Sync

Use iCloud, Google Drive, Dropbox, Direct to Mac, REST API, Home Assistant, or Nextcloud / WebDAV.

Convert Tool

Convert Apple Health XML to CSV, JSON, or Excel directly in your browser.

OpenClaw Integration

Use the Mac app’s local API with OpenClaw for daily briefs and natural-language health questions.

API Reference

Read-only localhost API for developers, scripts, and custom integrations.

AI Analysis

Use OpenAI, Anthropic, OpenRouter, Ollama, or LM Studio to analyze your health data.

How the Product Works Today

  • iPhone app: export Apple Health data manually, sync it to a provider, and configure the Home Screen widget.
  • Mac app: import Apple Health XML, download synced exports from the Mac Sync tab, run analysis, and expose a local API for integrations.
  • Widget: choose whether it starts Quick Sync, Quick Export, or opens the Sync screen. The widget action is configured in iPhone Settings, while the displayed provider comes from the iPhone Sync tab.

Recommended Paths

Fastest manual path

Export XML on iPhone, AirDrop it to Mac, then import and analyze.

Best ongoing setup

Configure sync on iPhone, use the widget for quick refreshes, then pull the latest export from the Mac Sync tab.

Best for integrations

Import on Mac, select a dataset for integrations, enable the local API, then use OpenClaw or scripts.
