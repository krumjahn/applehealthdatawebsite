Mac App Overview

The Mac app is your central hub for working with Apple Health data. Import exports from your iPhone, run AI-powered analysis with your choice of provider, serve data through the local API, and connect to OpenClaw for natural language health queries.

All processing happens locally on your Mac. Your health data stays on your device unless you explicitly choose to send it to an AI provider.

Key Features

Multi-Format Import

Import Apple Health exports in XML, CSV, and JSON formats.

AI Analysis

Analyze data with OpenAI, Anthropic, OpenRouter, Ollama, or LM Studio.

Local API

Read-only API on port 8765 for scripts, automations, and integrations.

OpenClaw

Natural language health queries and daily briefs powered by OpenClaw.

Cloud Backup

Sync backups to iCloud or Google Drive for safekeeping.

Menu Bar Status

Always-visible menu bar indicator for sync status and quick actions.

System Requirements

  • macOS 13 Ventura or later
  • Apple silicon (M1 or later) or Intel Mac
  • An Apple Health export file, or the iPhone app for WiFi Sync

Installation

Download the Mac app from the App Store. Open it and you are ready to import your first health data export.

App Sections

Dashboard

Your at-a-glance overview of imported health metrics. See trends for steps, heart rate, sleep, workouts, and more. The dashboard updates automatically when new data is imported or synced.

Analysis

Run AI-powered analysis on your health data. Choose your preferred AI provider, select the metrics and date range you want to analyze, and get insights in plain language. Supports OpenAI (GPT-4o, o1, o3), Anthropic (Claude), OpenRouter, and local models via Ollama or LM Studio.

Export

Convert your imported health data into different formats. Export to CSV for spreadsheets, JSON for developers, or XLSX for Excel. Filter by date range and health category before exporting.

API / OpenClaw

Configure the local API server and OpenClaw integration. The API runs on localhost:8765 and provides read-only access to your health data for scripts, shortcuts, and third-party tools. OpenClaw connects your health data to a conversational AI interface for natural language queries.

Settings

Manage your API keys for AI providers, configure sync preferences, set up cloud backup destinations (iCloud, Google Drive), and customize which health categories to track.

Getting Data into the Mac App

There are three ways to get your Apple Health data into the Mac app:

Method How It Works Best For
WiFi Sync Direct transfer from iPhone over your local network Regular syncs, selective data, fastest setup
AirDrop Export from the iPhone app, then AirDrop the file to your Mac One-time transfers, full exports
File Import Drag and drop an export file (XML, CSV, or JSON) into the Mac app Files from email, cloud storage, or other sources

For the smoothest experience, WiFi Sync is recommended. It handles everything automatically and lets you choose exactly which health categories to transfer.
