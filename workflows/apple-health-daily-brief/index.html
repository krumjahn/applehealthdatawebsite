Workflow

Apple Health Daily Brief

A short daily summary (status → what changed → suggestions → missing data) generated from your local Apple Health dataset on Mac.

What you get

  • Steps vs your 7-day baseline
  • Sleep hours + missing-sleep detection
  • Workouts count + duration
  • Simple signals (e.g., activity below baseline)
  • 3 suggested actions for today

Setup (5 minutes)

  1. Install the Mac app: Health Data Export & AI Analyzer
  2. Export Apple Health data from iPhone and import it into the Mac app. (Guide: How to export Apple Health data)
  3. Install the OpenClaw skill from ClawHub: apple-health-export-analyzer
  4. Confirm the localhost API is running in the Mac app (OpenClaw / API tab). Then try:
curl http://127.0.0.1:8765/openclaw/status

The prompt (copy/paste)

Use this exact prompt to get a consistent “daily brief” output:

Use the "health-analyzer-mac-local" skill for this request.
Give me my daily health brief for today and 3 suggestions.

Tip: change “today” to a specific date like 2026-03-19.

Make it a habit

The daily brief is most valuable when it’s frictionless. Two easy patterns:

Morning check-in

Ask for yesterday + today. You’ll spot sleep gaps and activity drops quickly.

After workouts

Ask “compare today’s workout load vs my baseline” to avoid overreaching.
